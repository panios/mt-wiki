# Tilde MT

European (Latvian) language-technology vendor with 30+ years in MT,
positioned for EU-language coverage and EU-data-residency posture
([Tilde](https://tilde.ai/translate/tilde-machine-translation/features/),
 [EC WEB-T](https://website-translation.language-tools.ec.europa.eu/automated-translation_en)).
Tilde MT is one of the named MT providers integrated with the European
Commission's WEB-T website-translation framework
([EC WEB-T](https://website-translation.language-tools.ec.europa.eu/automated-translation_en)).

For market-context see [[../comparisons/mt-engines]].

## Shape

- **Origin:** Tilde, a 30+ year European language-tech company
  building NMT, LLM-driven chatbots, speech tech, localisation and
  terminology tooling
  ([EC WEB-T](https://website-translation.language-tools.ec.europa.eu/automated-translation_en)).
- **Distribution:** Direct API + plugins for Trados (RWS appstore)
  and MemoQ; embeddable as an iframe on customer sites; integrated
  with EU WEB-T plugins (Drupal, Joomla, WordPress, WooCommerce)
  ([Tilde](https://tilde.ai/translate/tilde-machine-translation/features/),
   [EC WEB-T](https://website-translation.language-tools.ec.europa.eu/automated-translation_en)).

## Capability mix

- 40+ MT engines plus large language models trained on general data,
  ready to use immediately
  ([Tilde](https://tilde.ai/translate/tilde-machine-translation/features/)).
- File translation preserving formatting (doc, xlsx, pdf, pptx,
  images, interchange formats)
  ([Tilde](https://tilde.ai/translate/tilde-machine-translation/features/)).
- Website translation by URL; CAT-tool plugin for Trados / MemoQ
  ([Tilde](https://tilde.ai/translate/tilde-machine-translation/features/)).
- LLM-augmented features: smart summarisation; translations in three
  registers (Academic / General / Professional)
  ([Tilde](https://tilde.ai/translate/tilde-machine-translation/features/)).
- Glossaries that can be defined and shared across the organisation
  for terminology consistency
  ([Tilde](https://tilde.ai/translate/tilde-machine-translation/features/)).

## Deployment

- **Default:** Cloud-hosted (vendor SaaS), with SSL and a stated "no
  storage / no analysis" data-handling policy — translated data
  deleted immediately after translation
  ([Tilde](https://tilde.ai/translate/tilde-machine-translation/features/)).
- **On-prem:** Tilde MT can be deployed on customer infrastructure,
  either in private cloud or fully on-prem
  ([Tilde](https://tilde.ai/translate/tilde-machine-translation/features/)).
- **Custom MT engines:** trained on customer data, tailored to
  industry / terminology / style
  ([Tilde](https://tilde.ai/translate/tilde-machine-translation/features/)).
- **White-labeling:** branded translation portal (logo, colours,
  fonts) and embeddable iframe for integration into customer
  applications
  ([Tilde](https://tilde.ai/translate/tilde-machine-translation/features/)).

## Where it fits / doesn't (`mode/advise`)

- **Fits:** EU-headquartered clients prioritising EU-vendor and
  EU-data-residency posture (especially public-sector clients
  participating in the Digital Europe Programme via WEB-T); mid-sized
  organisations needing a branded translation portal; Trados / MemoQ
  shops looking to add an MT layer through an existing CAT-tool
  plugin path
  ([Tilde](https://tilde.ai/translate/tilde-machine-translation/features/),
   [EC WEB-T](https://website-translation.language-tools.ec.europa.eu/automated-translation_en)).
- **Doesn't fit:** Workflows that need the broadest possible
  language coverage (Google Cloud, RWS Edge cover more); deployments
  with FDA CFR Part 11 / FedRAMP / HIPAA hard requirements (not
  documented in the source); North-American defense procurement
  (vendor origin signals against this).

## Source-quality caveat

The Tilde page is vendor self-marketing; the EC WEB-T page is a
neutral integration directory but limited to bullet-point summaries
provided by each vendor. The "no storage / no analysis" data-handling
claim is on the cloud-hosted product — the on-prem deployment
collapses the question entirely. No third-party benchmarks are in
either source.

## Open questions

- Compliance certifications (ISO 27001 / SOC 2 / GDPR formal
  certification status, beyond stated GDPR alignment).
- Per-language quality benchmarks vs. DeepL / Google for the EU
  language pairs Tilde is positioned for.
- The "40+ engines" claim — are these in-house Tilde-trained engines
  or a meta-platform that fronts third-party engines? The source
  reads as in-house but is ambiguous.
- On-prem hardware sizing and update path.
